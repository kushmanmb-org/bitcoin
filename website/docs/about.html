About Kushmanmb.org

Our Mission

Kushmanmb.org is dedicated to advancing Bitcoin Core development with a strong emphasis on security, privacy, and best practices in software engineering. We build upon the foundation of Bitcoin Core while implementing enhanced security measures and modern development workflows.

What We Do

We focus on:

  • Secure Development: Implementing industry-leading security practices throughout the development lifecycle
  • Code Quality: Maintaining high code quality standards with automated testing and code review
  • Innovation: Exploring new approaches to Bitcoin development while maintaining compatibility
  • Documentation: Providing comprehensive documentation for developers and users
  • Community: Building an engaged community of developers and contributors

Our Approach

Security First

Modern CI/CD

We use modern continuous integration and deployment practices:

  • Self-hosted runners for enhanced security
  • Automated testing on every change
  • Security scanning as part of CI pipeline
  • Automated dependency updates

Privacy Protection

We take privacy seriously:

  • Comprehensive .gitignore to prevent accidental exposure
  • No tracking or analytics on this website
  • Secure credential storage practices
  • Data minimization principles

Technology Stack

  • Core: C++17, Bitcoin Core codebase
  • Build System: CMake
  • Testing: Python functional tests, C++ unit tests
  • CI/CD: GitHub Actions with self-hosted runners
  • Security: CodeQL, OWASP Dependency Check, Snyk
  • Package Management: Maven, GitHub Packages

Project Principles

  1. Security: Security is paramount in everything we do
  2. Transparency: Open source, open development
  3. Quality: High standards for code quality and testing
  4. Privacy: Respect for user and developer privacy
  5. Community: Welcoming and inclusive community
  6. Documentation: Clear, comprehensive documentation

Based on Bitcoin Core

This project builds upon Bitcoin Core, the reference implementation of Bitcoin. We maintain compatibility while implementing additional security features and development practices.

Bitcoin Core is:

  • Open source (MIT License)
  • Peer-reviewed and battle-tested
  • The most widely used Bitcoin implementation
  • Maintained by a global community of developers
